# RibbitXDB Database Viewer

A simple database viewer for the [RibbitXDB](https://github.com/ribbitx/ribbitxdb) engine

## Core functions

* Open database
* View tables and views
* View columns and data types/constraints
* View table schema and create script
* Generate queries for table
* Dropping tables and views
* Query execution, both selected and whole query
* Query syntax highlighting
* Loading and saving SQL
* Exporting results to CSV

### Requirements

```requirements
ribbitxdb~=1.1.7
PySide6-Essentials~=6.10.1
platformdirs~=4.5.1
pyinstaller~=6.17.0
```
